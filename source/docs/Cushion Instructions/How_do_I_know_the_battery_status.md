## How do I know the battery status?

#### From the indicator

| Working Status | Battery Status | Indicator |
|---|---|---|
| Charging | Low battery | Solid Red |
|  | Charging | Flashing red |
|  | Fully Charged | Light off |
| In Use | Normal | Solid Blue |
