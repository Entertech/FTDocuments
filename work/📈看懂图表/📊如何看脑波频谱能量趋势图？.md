
# 📊如何看脑波频谱能量趋势图？

脑电波频段能量占比的变化趋势体现了冥想过程中精神状态的变化。

| 名称 | 冥想状态 |
| :---: | :---: |
| γ波 | 资深经验的冥想人士在冥想过程中该频段可能出现较高能量 |
| β波 | 冥想过程中，在专注度较高时该频段能量较高 |
| α波 | 冥想过程中，由于闭眼和放松，该频段能量会比平时有所增加 |
| θ波 | 具有一定冥想经验的人在冥想过程中可以出现较高的θ波，但同时又保持清醒 |
| δ波 | 冥想过程中一般没有太大变化 |
